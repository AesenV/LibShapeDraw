glApply

void glApply(double alphaScale)
Convenience method to update the OpenGL state to use this color, with the alpha value scaled by a specified factor. The Color instance remains unchanged by the factor. The final alpha value passed to OpenGL is clamped to [0.0, 1.0].

Equivalent to c.copy().scaleAlpha(alphaScale).glApply(), without the extra Color instance creation.

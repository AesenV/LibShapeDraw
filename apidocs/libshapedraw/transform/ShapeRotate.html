Description copied from interface: Animates
Start a finite animation on this object's properties, gradually changing them to match the specified properties over the specified interval.

After starting an animation, using any other method that modifies the properties is a bad idea, as the animation will be frequently overwriting them. Either wait for the animation to complete or use Animates.animateStop() to halt the animation early.

If an animation is already active when this start method is called, animateStop is automatically called before starting the new animation.

animateStartLoop

public ShapeRotate animateStartLoop(Double toAngleDegrees,
                                    boolean reverse,
                                    long durationMs)
Description copied from interface: Animates
Start a looping animation on this object's properties, gradually changing them to match the specified properties over the specified interval.

The animation loops indefinitely, going back and forth between the original properties and the specified properties.

After starting an animation, using any other method that modifies the properties is a bad idea, as the animation will be frequently overwriting them. Use Animates.animateStop() to halt the animation first.

If an animation is already active when this start method is called, animateStop is automatically called before starting the new animation.

Specified by:
animateStartLoop in interface Animates<Double>
Parameters:
toAngleDegrees - the properties to gradually change to.
reverse - if true, fade back to the original properties each time the animation loops. If false, jump directly back to the original properties each time.
durationMs - interval in milliseconds of each cycle, >= 0.
Returns:
the same object, modified in-place.

Internal class. Mods using the LibShapeDraw API can safely ignore this. Rather, instantiate LibShapeDraw.

This is a ModLoader mod (also compatible with Forge/FML) that links itself to the internal API Controller, providing it data and events from Minecraft. This class does the bare minimum of processing before passing these off to the controller. I.e., this class is a thin wrapper for Minecraft used by LibShapeDraw.

As a wrapper, all direct interaction with Minecraft objects passes through this class, making the LibShapeDraw API itself clean and free of obfuscated code. (There is a single exception: LSDModDirectory.getMinecraftDir.)
